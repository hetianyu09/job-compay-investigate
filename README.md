# Job Company Investigator Skill

Codex skill for investigating Chinese company background and software/IT role risk.

## Install

Copy the `job-company-investigator` folder into your local Codex skills directory:

```text
C:\Users\<your-name>\.codex\skills\job-company-investigator
```

Then restart or refresh Codex and invoke:

```text
Use $job-company-investigator to investigate <company name>.
```

## What It Covers

- Company background, main business, founding time, scale, social insurance headcount, financial condition, and labor disputes
- Internet-company classification and product analysis
- Software/IT role risks, including salary structure, probation conversion, interview experience, work intensity, overtime, layoffs, and severance
- Multi-platform workplace reputation checks across Maimai, Douyin, Xiaohongshu, Douban, Zhihu, and other social platforms
